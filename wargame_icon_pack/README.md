# Wargame Icon Pack

Small transparent tactical icons for a modern naval/air/cyber wargame UI.

Contents:
- png_128/: 128x128 transparent PNG icons
- svg/: matching scalable SVG icons

Design:
- Simple top-down / tactical-map style
- Dark charcoal base shapes
- Light gray accents
- Purple for cyber/EW
- Orange for damage / kinetic warnings
- No text labels embedded in icons

Suggested usage:
- Use PNGs for quick prototyping.
- Use SVGs for scalable in-game UI.
- Apply color tinting in-engine for allegiance:
  - Blue = friendly
  - Red = hostile
  - Gray = unknown
  - Yellow = suspicious
  - White = civilian / neutral
  - Purple = cyber/EW
